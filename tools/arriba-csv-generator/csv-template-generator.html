Gerador de CSV — Layout Padrão (Dock)

Gera contrato.csv, financiado.csv e parcela.csv com todas as colunas. Vermelho = obrigatório. Preto = opcional (vai vazio ou pelo “Defaults”).

Config geral

CPF/CNPJ — FINANCIADO (opcional)

Obrigatórios — CONTRATO

Qualquer coluna do header de contrato pode entrar aqui. Se não informar, sai vazio.

Obrigatórios — FINANCIADO

Os obrigatórios sempre sobrescrevem defaults. O resto entra do jeito que você digitar.

Obrigatórios — PARCELA

Vl_Saldo sempre sai igual ao Vl_Original (regra simples, bem estável).

Separador: ; — Quebra de linha: Windows (CRLF)
