Operação

Passo a passo para o cliente

  1. Acessar a base de Homologação. O fluxo começa na base de homologação do DataCob.
  2. Abrir o menu de ajuda. Na tela inicial, clicar no ícone de ajuda (?) e selecionar Checklist versão.
  3. Validar os itens do checklist. O cliente deve conferir se os itens estão funcionando corretamente.
  4. Finalizar checklist. Depois de validar, clicar em Finalizar Checklist.
  5. Escolher data e hora da virada. Na tela de aprovação, selecionar a data/hora para a virada de versão e confirmar em Sim.
  6. Confirmar mensagem de sucesso. Após o agendamento, o sistema exibe mensagem de checklist finalizado e agendamento realizado com sucesso.
Regras principais

Regras do agendamento automático

Mesmo dia

Para agendar no mesmo dia, o checklist deve ser finalizado até as 17h. Depois disso, fica pendente para aprovação/atuação do suporte.

Dias permitidos

O agendamento automático deve ocorrer de segunda a quinta-feira, entre 20h e 00h.

Sexta, sábado e domingo

Se a data cair nesses dias, o sistema alerta e o suporte deve orientar/agendar no próximo dia útil.

Horário operacional

Antes das 20h é considerado horário operacional e pode impactar a cobrança; o agendamento fica pendente para atuação do suporte.

Atenção

Cancelamento de agendamento

Se o cliente cancelar o agendamento, ele não conseguirá realizar o agendamento automático novamente até que uma nova atualização/versão seja enviada. Nesse caso, deve entrar em contato com o suporte PH3A para nova tratativa.

Respostas predefinidas relacionadas

@Respostas para o cliente atualizar versão - (Checklist Versão)

Use quando o cliente precisar ser orientado a executar o checklist e agendar a atualização.

Informamos que a versão do DataCob em seu ambiente de produção está DESATUALIZADA.

Solicitamos que efetue o checklist e agende o mais breve possível a atualização do sistema, pois assim mantemos o sistema sempre apto para executar todas as suas atribuições.

Nessas atualizações podem conter:
- Melhorias de usabilidade;
- Correção de falhas e melhor confiabilidade;
- Novas funcionalidades.

Lembrando que os itens da versão estão disponíveis no Histórico de Versões no menu de ajuda do DataCob. Não se esqueça de efetuar o Checklist da Versão para realizar o agendamento da atualização.

CHECKLIST VERSÃO

Passo 1: Na tela inicial, clicar no ícone (?) e depois Checklist versão.
Passo 2: Na tela do checklist, após validar que todos os itens estão funcionando corretamente, clicar em Finalizar Checklist.
Passo 3: Escolher a melhor data e hora para realizar a virada de versão e confirmar em Sim.

Se não conseguir realizar o agendamento, informe a data e horário desejados para que possamos apoiar a atualização do ambiente.

@Respostas para o cliente quer atualizar versão - (Resposta do agendamento)

Use quando a data/horário já estiverem definidos e o suporte for confirmar o agendamento.

Boa tarde, a todos,

Support Copilot

Como o Copilot deve usar este artigo

  • Classificar tickets com palavras-chave de versão como Tipo Freshdesk: Versão.
  • Recomendar DataCob como produto quando o contexto mencionar checklist, homologação ou virada de versão.
  • Sugerir a resposta predefinida de checklist quando o cliente ainda não agendou.
  • Sugerir a resposta de agendamento quando a data e o horário já estiverem definidos.
  • Adicionar este artigo na aba Base para o analista abrir, copiar e validar.
Checklist para o analista: - Confirmar se o cliente está no ambiente de homologação; - Orientar acesso ao ícone de ajuda (?) e à opção Checklist versão; - Confirmar se o checklist foi finalizado com sucesso; - Validar data e horário pretendidos para a virada; - Se for mesmo dia, confirmar se o checklist foi finalizado até 17h; - Se for sexta, sábado ou domingo, orientar que o suporte deverá atuar para efetivar no próximo dia útil; - Se o agendamento foi cancelado, orientar que será necessário contato com o suporte para nova tratativa.
