Boleto Base64 → PDF

Cole o Base64 do boleto (ou a resposta JSON da API que contém o campo em Base64), visualize o PDF na hora e baixe. Feito para o time de suporte quando o boleto chega como Base64 no chamado. Processamento 100% local no navegador — nada é enviado para servidor.

Passo 1

Cole o Base64

Aceita Base64 puro, Data URI (data:application/pdf;base64,…) ou um JSON com o Base64 dentro (ex.: { "boleto": "JVBER..." }) — o campo é localizado automaticamente.
