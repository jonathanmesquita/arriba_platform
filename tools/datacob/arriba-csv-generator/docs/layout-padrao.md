# Documentação rápida — Layout Recepção de Contratos

Arquivo base: `Layout Recepcao de Contratos.xlsx`

## Fluxo
1. Escolha o layout **01-Layout Recepção de Contratos - Avançado**.
2. Selecione as tabelas: Contrato, Financiado e/ou Parcela.
3. Preencha os dados obrigatórios.
4. Use os campos avançados apenas quando necessário.
5. Valide, confira o preview e baixe o CSV.

## Campos obrigatórios da tela beta
- Quantidade de registros
- Cliente
- Contrato inicial
- Sufixo do contrato
- Fase
- Documento
- Vencimento
- Valor original
- Tipo parcela
- Número da parcela

## Roadmap
Em versão futura, o cliente poderá subir dados crus. A ferramenta fará a leitura, separação e transformação automática para Contrato.csv, Financiado.csv e Parcela.csv.
