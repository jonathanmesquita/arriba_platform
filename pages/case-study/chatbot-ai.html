Fluxo

IA útil mesmo quando a API falha.

O frontend envia a mensagem e o modo selecionado para a API. O backend tenta responder com IA; se houver erro de quota, indisponibilidade ou falha externa, retorna uma resposta local útil.

  • Modos: DataCob, SQL, DevOps e Geral.
  • Frontend: JavaScript modular e UI persistente.
  • Backend: Express + rota POST /chat.
  • Fallback: base local com palavras-chave e respostas técnicas.
