Layout Recepção de Contratos

Documentação rápida para apoiar o uso do Gerador CSV DataCob beta.

Como usar

  1. Escolha o layout 01-Layout Recepção de Contratos - Avançado.
  2. Selecione as tabelas: Contrato, Financiado e/ou Parcela.
  3. Preencha os dados obrigatórios.
  4. Use campos avançados apenas quando necessário.
  5. Valide, confira o preview e baixe o CSV.

Campos obrigatórios da tela beta

  • Quantidade de registros
  • Cliente
  • Contrato inicial
  • Sufixo do contrato
  • Fase
  • Documento
  • Vencimento
  • Valor original
  • Tipo parcela
  • Número da parcela
