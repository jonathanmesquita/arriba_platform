Cadastro de Novas Carteiras

Treinamento: Cadastro de Novas Carteiras DataCob

📋 Visão Geral

Este treinamento orienta o cliente a compreender o fluxo básico de cadastro de uma nova carteira no DataCob. O processo apoia a configuração inicial de Grupo, Cliente e Fase, permitindo organizar contratos, estratégias de cobrança e parâmetros operacionais.

🛠️ Passo a Passo Prático

Acessar o ambiente de Homologação

O que fazer: Entre no ambiente de homologação do DataCob antes de alterar parâmetros em produção.

Elemento na tela: Indicador “Homologação” no topo do sistema.

O que esperar: O usuário acessa uma base segura para testes.

Abrir Parâmetros

O que fazer: No menu superior, acesse Parâmetros e localize as opções de visualização/cadastro.

Elemento na tela: Menu horizontal com as opções Operações, Recepções, Exportações, Distribuição, Parâmetros e Relatórios.

O que esperar: O sistema exibe as rotinas de parametrização da carteira.

Configurar Grupo / Cliente / Fase

O que fazer: Cadastre ou valide o Grupo, Cliente e Fase que representarão a nova carteira.

Elemento na tela: Abas ou colunas identificadas como Grupo, Cliente e Cliente Fase Empresa.

O que esperar: A carteira fica estruturada para receber contratos e regras de cobrança.

Validar regras complementares

O que fazer: Verifique régua, conta, boleto, ocorrência, política de cálculo, acordo e recibo.

Elemento na tela: Telas de parâmetros e cadastros relacionados à carteira.

O que esperar: A carteira fica pronta para testes operacionais.

⚠️ Pontos de Atenção

  • Nunca aplicar alteração direto em produção sem validação prévia.
  • Confirmar se a regra vale para Grupo, Cliente, Fase ou carteira específica.
  • Registrar evidências dos testes antes de liberar para operação.
  • Encaminhar para o suporte/DEV apenas se o comportamento divergir da parametrização validada.
