        Roadmap
        
    Passo a passo do projeto
    
        
    
          
    1. MVP manual
    Formulario na Arriba Platform para colar chamado e gerar analise.
    
          
    2. API Node
    Endpoint no Render para IA, fallback local e resposta estruturada.
    
          
    3. Freshdesk
    Buscar ticket, renderizar template, adicionar nota interna e sugerir resposta predefinida.
    
          
    4. Suporte preditivo
    Detectar recorrencia por produto, rotina e tipo de solicitacao.
    
        
    
      
    
    
    
  
    


  
    
    
    
      Arriba Auth
      
    Acesso protegido
    
      
    Entre para consultar tickets, base Freshdesk e contexto interno do suporte.
    
      
    
        
        
        
        
        
    
        
      
    
      Quando a sessão expirar, o Copilot pausa a ação e pede login novamente.
