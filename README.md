# Arriba Platform

Arriba Platform é uma aplicação web moderna e responsiva para controle de produtividade e erros técnicos no dia a dia. Inspirada visualmente no site da Oracle e funcionalmente no site do GTA VI da Rockstar Games, esta plataforma oferece busca integrada, assistente virtual (chatbot), documentação SQL e integração com CRM.

![preview](https://user-images.githubusercontent.com/yourusername/arriba-platform/preview.png)

## 🚀 Funcionalidades

- 🔍 Busca inteligente por seções
- 💬 Chatbot técnico simulado
- 🌐 Suporte a múltiplos idiomas (PT, EN, ES)
- 🎨 Alternância entre tema claro e escuro
- 📚 Acesso à documentação SQL
- 🧩 Integração com CRM (Datacob)

## 🌍 Publicação via GitHub Pages

1. Crie um repositório no GitHub.
2. Faça upload dos arquivos do projeto.
3. Vá em **Settings > Pages**, selecione `main` como branch e `/ (root)` como diretório.
4. O projeto ficará disponível em `https://jonathanmesquita.github.io/arriba_platform/`.
